## `Boolean`问题
* 注意以下输出
* 建议用三目运算符

```js
//切记这样无法转换
Boolean('false')//true
Boolean('true')//true
```

## 表单多对象赋值
```js
//我需要创建person_B,属性除了name 其它都一样,还需要增加一个weight属性
    const {name, ...otherProps} = person_A
```

## 字符串长度问题
* 字符串在设计上是不可变的， 但可以将引用重新分配给新的字符串值
* 如果要缩短字符串，可以使用String slice（）//切割方法。
